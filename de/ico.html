Binary.com ICO FAQ

Häufig gestellte Fragen

Was ist ein ICO?

Ein „Initial Coin Offering” (oder kurz „ICO”) ist eine Geldbeschaffungs-Methode, die einer Börseneinführung weitgehend ähnlich ist. Bei einer ICO, werden Kryptowährungen so wie „Colored Coins” oder „Ethereum Token” anstelle von Aktien ausgestellt.

Warum will Binary.com ein ICO durchführen?

Binary.com ist ein etabliertes Unternehmen, das in 1999 gegründet wurde. Der Wachstum unseres Unternehmens hat uns, sowie im Fall der meisten erfolgreichen Unternehmen, dazu verleitet eine Börseneinführung zu erwägen. Allerdings glauben wir, dass die Börsen im kommenden Jahrzehnt schrittweise mit Kryptowährungs-Börsen ersetzt werden werden, und Börseneinführungen schrittweise mit ICOs (Initial Coin Offering) ersetzt werden werden. Das ist auch der Grund, weswegen wir uns dazu entschieden haben eine ICO anstelle einer Börseneinführung durchzuführen.

Wie funktioniert die ICO?

Binary.com wird eine Anzahl an Blockchain Token für Investoren anbieten. Diese Token können in diversen Formen auf unterschiedlichen Blockchains herausgegeben werden, so wie die „Colored Coins” auf der Bitcoin Blockchain und „ERC20” Token auf der Ethereum Blockchain.

Unabhängig von der Art des Tokens, verschafft Ihnen jeder Token zwei Rechte:

  1. Anspruch darauf, Zahlungen, die Dividenden entsprechen, zu erhalten; angezahlt auf Binary Group Ltd. Stammaktien
  2. Das Recht, Token in Stammaktien umzuwandeln

Wie werden die Token angeboten?

Die Token werden über einen Zeitraum von einem Monat angeboten. Das genaue Datum der ICO ist noch nicht bestimmt, aber wir zielen auf das vierte Quartal 2017 hin. Der ICO-Preis und die Auktions-Methode werden in Kürze bekanntgegeben.

Wer kann die Token kaufen?

Token werden nur an Binary.com Kunden angeboten, die ein Echtgeld-Konto haben und die unseren Konto-Authentifizierungsprozess abgeschlossen haben.

Ist die ICO in allen Ländern erhältlich?

Die ICO wird in einer Reihe von Ländern auf qualifizierte oder professionelle Investoren, in Übereinstimmung mit den Regeln und Vorschriften von solchen Ländern, beschränkt werden.

Wie kann ich Ihre Token kaufen?

Sie können unsere Token in Verwendung der Gelder in Ihrem Binary.com Konto erwerben. Wir werden bald native BTC (Bitcoin) und ETH (Ethereum) Konto Einrichtungen einführen, um Ihre Teilnahme am ICO zu ermöglichen.

Wie erfahre ich mehr über ICOs und Kryptowährungen?

Um mehr über Bitcoin zu erfahren, empfehlen wir, dass Sie ein Bitcoin-Wallet mit Blockchain eröffnen und Bitcoins mit einem Ortsamt in Ihrem Land erwerben.

Um mehr über Ethereum zu erfahren, empfehlen wir, dass Sie das Metamask-Plugin für Chrome installieren, das Ihnen erlauben wird, Ihre Bitcoin zu Ether zu konvertieren.

Um zu erfahren, wie man Kryptowährungs-Vermögenswerte handelt, empfehlen wir, dass Sie ein Konto mit dem US-amerikanischen Kryptowährungs-Austausch POLONIEX eröffnen.
