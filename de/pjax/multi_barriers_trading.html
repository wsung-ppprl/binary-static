Asiatische Optionen werden abgewickelt, indem der letzte Tick mit dem durchschnittlichen Kassakurs während des Zeitraums verglichen wird.

Wenn Sie „Asiatischer Anstieg" auswählen, erhalten Sie eine Auszahlung, wenn der letzte Tick höher als der Durchschnitt der Ticks ist.

Wenn Sie „Asiatischer Fall" auswählen, erhalten Sie eine Auszahlung, wenn der letzte Tick niedriger als der Durchschnitt der Ticks ist.

Wenn Sie „Gleich" auswählen, erhalten Sie eine Auszahlung, wenn die letzte Stelle des letzten Ticks gleich Ihrer Voraussage ist.

Wenn Sie „Verschieden" auswählen, erhalten Sie eine Auszahlung, wenn die letzte Stelle des letzten Ticks nicht mit Ihrer Voraussage übereinstimmt.

Wenn Sie „Schließt zwischen" auswählen, erhalten Sie eine Auszahlung, wenn der Schlusskurs genau über der Unteren Schwelle UND genau unter der Oberen Schwelle verharrt.

Wenn Sie „Schließt außerhalb" auswählen, erhalten Sie eine Auszahlung, wenn der Schlusskurs ENTWEDER genau über der Oberen Schwelle ODER genau unter der Unteren Schwelle bleibt.

Wenn Sie „Gerade" auswählen, erhalten Sie eine Auszahlung, wenn die letzte Stelle des letzten Ticks eine gerade Zahl (z.B. 2, 4, 6, 8 oder 0) ist.

Wenn Sie „Ungerade" auswählen, erhalten Sie eine Auszahlung, wenn die letzte Stelle des letzten Ticks eine ungerade Zahl (z.B. 1, 3, 5, 7, oder 9) ist.

Wenn Sie „Höher" auswählen, erhalten Sie eine Auszahlung, wenn der Kurs zum Vertragsende genau über der Schwelle liegt.

Wenn Sie "Tiefer" auswählen, erhalten Sie eine Auszahlung, wenn der Kurs zum Ende des Kontrakts genau unter der Schwelle liegt.

Wenn Sie „Über" auswählen, erhalten Sie eine Auszahlung, wenn die letzte Stelle des letzten Ticks größer als Ihre Voraussage ist.

Wenn Sie „Unter" auswählen, erhalten Sie eine Auszahlung, wenn die letzte Stelle des letzten Ticks weniger als Ihre Voraussage ist.

Wenn Sie "Steigt" auswählen, erhalten Sie eine Auszahlung, wenn der Kurs zum Auslauf des Kontrakts genau über dem Kurs bei Beginn des Kontrakts liegt.

Wenn Sie „Fällt" auswählen, erhalten Sie eine Auszahlung, wenn der Schlusskurs genau unter dem Anfangskurs liegt.

Sie gewinnen die Auszahlung, wenn der Marktkurs mit der Dezimalstelle schließt, die Sie ausgewählt haben.

Sie gewinnen die Auszahlung, wenn der Marktkurs nicht auf der Dezimalstelle schließt, die Sie ausgewählt haben.

Wenn Sie „steigt" auswählen, erhalten Sie eine Auszahlung, wenn der Marktpreis höher als der Kurs bei Kontraktbeginn ist.

Wenn Sie „Fällt" auswählen, erhalten Sie eine Auszahlung, wenn der Marktpreis niedriger als der Kurs bei Kontraktbeginn ist.

Wenn Sie „Bleibt zwischen" wählen, erhalten Sie eine Auszahlung, wenn der Kurs zu irgendeinem Zeitpunkt während der Kontraktlaufzeit zwischen der Oberen Schwelle oder der Unteren Schwelle bleibt (ohne sie zu berühren).

Wenn Sie „Geht außerhalb" auswählen, erhalten Sie eine Auszahlung, wenn der Kurst während der Kontraktlaufzeit zu einem bestimmten Zeitpunkt entweder die ObereSchwelle oder die Untere Schwelle berührt.

Wenn Sie "Steigt" auswählen, erhalten Sie eine Auszahlung, wenn der Kurs zum Auslauf des Kontrakts genau über dem Kurs bei Beginn des Kontrakts liegt.

Wenn Sie „Fällt" auswählen, erhalten Sie eine Auszahlung, wenn der Schlusskurs genau unter dem Anfangskurs liegt.

Wenn Sie „Berührt" wählen, erhalten Sie eine Auszahlung, wenn der Kurs die Schwelle zu irgendeinem Zeitpunkt während der Kontraktlaufzeit berührt.

Wenn Sie „Erreicht nicht" wählen, erhalten Sie eine Auszahlung, wenn der Kurs die Schwelle während der Kontraktdauer zu keinem Zeitpunkt berührt.
