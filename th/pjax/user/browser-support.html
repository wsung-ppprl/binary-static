ปัญหาการเข้าสู่ระบบ

หากท่านไม่สามารถเข้าใช้งานบัญชีของท่าน โปรดตรวจสอบคุกกี้ของเบราเซอร์ท่าน เว็บไซต์ของเราใช้คุกกี้ที่เก็บอยู่ในบัญชีผู้ใช้งานและรหัสผ่านของท่าน (ที่มีการเข้ารหัส) ในเครื่องคอมพิวเตอร์ของท่านในขณะที่ท่านใช้งานระบบซื้อขาย

เพื่อติดตั้งคุกกี้ โปรดปฏิบัติตามขั้นตอนข้างล่างสำหรับเวอร์ชันของเบราเซอร์ที่ท่านใช้งาน

Binary (C.R.) S.A.

Chrome 51+

  • Go to "Chrome Menu (Customize and control Google Chrome)" dropdown and select Settings.
  • Click on the Show advanced settings.
  • Click on the Content settings button under Privacy menu.
  • Select Allow local data to be set (recommended).
  • Click on the Done button to save your settings.
Binary (IOM) Ltd

Mozilla Firefox 47+

  • Open menu and select Preferences.
  • Click on the Privacy tab.
  • Set "Firefox will" to "Remember history".

Safari 9.x+

  • Go to the Safari menu and select Preferences.
  • Click on the Privacy tab.
  • Set "Block cookies:" to Never.
  • Click on the X button at the top right to save and close.
Binary (C.R.) S.A.

Opera 38+

  • Go to the Tools menu and select Preferences.
  • Click on the Advanced tab.
  • Select Cookies from the left column.
  • Select Accept cookies.
  • Click on the OK button to save and close.
Binary Ltd

Microsoft Internet Explorer 11+

  • Go to the Tools menu and select Internet Options.
  • Click on the Privacy tab.
  • Click on the Advanced button.
  • Select the Override automatic cookie handling.
  • Select Accept for both First-party Cookies and Third-party Cookies.
  • Click on the OK button to save and close.
Binary Ltd

Microsoft Edge 12+

  • Go to the More actions menu and select Settings.
  • Under Advanced Settings, click on the View advanced settings button.
  • Under the Cookies section, select the "Don't block cookies" option.
  • Click anywhere outside the menu to save and close.
