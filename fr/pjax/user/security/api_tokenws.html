Jeton d'API

Pour accéder à nos applications mobiles et aux autres applications tierces, vous devrez d'abord générer un jeton d'API.

  • Cliquez simplement sur « Créer » pour générer votre jeton, puis copiez-collez votre jeton dans l'application.
  • Choisissez le type de jeton d'API dont vous avez besoin, selon les fonctionnalités que vous souhaitez proposer.
